# State



## 描述

提供给PaaS及接入方调用，用于通过API获取游戏汇总数据



## 请求



### 请求参数

| 参数名      | 类型   | 是否必填 | 说明     |
| ----------- | ------ | -------- | -------- |
| instance_id | String | 是       | 实例ID   |
| data        | Object | 是       | 携带数据 |



### 响应参数

| 参数    | 类型   | 说明     |
| ------- | ------ | -------- |
| code    | Int    | 返回码   |
| message | String | 结果描述 |
| Data    | Object | 返回数据 |



### 响应示例

```json
{
    "code": 0,
    "message": "",
  	"data":{}
}
```
