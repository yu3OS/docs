# PaaS 游戏接入平台 文档

## 项目简介

PaaS 游戏接入平台（3OS 开放平台）为游戏开发者提供完整的游戏发布、接入和运维能力。开发者按平台要求提供**游戏服务端、游戏后台、H5 客户端**，审核通过后即可接入 PaaS 平台分发给多个接入方。

## 目录结构

```
docs/
├── BusinessAPI/          # 接入方（业务方）需要实现的 API
│   ├── Client/           # 客户端接入文档（Android/iOS WebView + JsBridge）
│   └── Server/           # 服务端接口文档（用户认证、积分系统等）
├── DevAPI/               # 开发者需要了解的 API
│   ├── 交互API/          # 平台提供给开发者的接口
│   └── PAAS回调API/      # 开发者需要实现的回调接口
└── 开发及接入流程/        # 完整的开发和接入指南
```
